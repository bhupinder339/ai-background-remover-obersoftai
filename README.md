# AI Background Remover & Image Enhancer

This project removes image background using AI and enhances image quality.

## Features
- AI-based background removal
- Image brightness & sharpness enhancement
- CPU-friendly (AWS EC2 compatible)

## Tech Stack
- Python
- rembg
- OpenCV
- Pillow

## How to Run
```bash
pip install rembg[cpu] pillow opencv-python
python3 bg_remove.py
python3 enhance.py
